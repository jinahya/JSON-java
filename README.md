## usages
### JRE 1.3
```xml
<dependency>
  <gropuId>com.github.jinahya</groupId>
  <artifactId>json-retrotranslated</artifactId>
  <classifier>retrotranslated13</classifier>
</dependency>
```
### JRE 1.4
```xml
<dependency>
  <gropuId>com.github.jinahya</groupId>
  <artifactId>json-retrotranslated</artifactId>
  <classifier>retrotranslated14</classifier>
</dependency>
```
